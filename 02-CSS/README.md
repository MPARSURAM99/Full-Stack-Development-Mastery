# Personal Portfolio (HTML & CSS Project)

This project is a **responsive personal portfolio website built using HTML and CSS**.  
The purpose of this project is to practice **semantic HTML, modern CSS layout techniques, responsive design, and UI styling** while building a professional developer portfolio.

This project represents an important step in my journey toward becoming a **Full Stack Developer**, focusing on strong **frontend fundamentals**.

---

# Project Overview

The portfolio website introduces my **professional profile, education, technical skills, projects, certifications, and contact information**.

The website includes multiple sections designed to present my work and skills in a clean and structured layout.

Main sections included in the project:

- Navigation Bar
- Hero / Introduction Section
- Education Timeline
- Skills Section with Circular Progress Bars
- Projects Showcase
- Certification Cards
- Contact Section
- Footer

The project focuses on creating a **structured, responsive, and visually appealing portfolio using only HTML and CSS**.

---

# Learning Outcomes

Through this project I practiced:

- Writing structured **semantic HTML**
- Designing layouts using **CSS Grid and Flexbox**
- Creating **responsive layouts**
- Building reusable UI components
- Implementing modern CSS techniques
- Designing a clean **developer portfolio interface**

This project helped strengthen my understanding of **frontend development fundamentals**.
