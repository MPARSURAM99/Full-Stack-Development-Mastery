# Personal Portfolio (HTML Project)

This project is a **basic personal portfolio website built using pure HTML**.  
The purpose of this project is to practice **semantic HTML, document structure, and content organization** while building a simple developer portfolio.
This project represents my **first step in mastering frontend development fundamentals**.

---

# Project Overview

The portfolio webpage contains essential sections that introduce my **professional profile, education, technical skills, and projects**.

The goal of this project was to:

- Practice **semantic HTML structure**
- Learn how to organize content properly in a webpage
- Understand **HTML elements and document hierarchy**
- Build a **simple portfolio layout without using CSS or JavaScript**

# Learning Outcomes

Through this project I practiced:

- Writing structured HTML documents
- Using semantic HTML elements
- Organizing content clearly
- Creating a portfolio layout using only HTML
